Unordered List
Items inside the list do not have a specific order or the order is not important.
Ordered List
Items inside the list have a specific order and/or the order of items in the list is important.
Description List
This is a special type of list where we have a term and description of the term.
Paragraphs
The browser will add a space before and after the content of the paragraph.
Emphasis
Screen readers will change their voice for this text. A string of emphasized, slightly more important text.
Superscript text
